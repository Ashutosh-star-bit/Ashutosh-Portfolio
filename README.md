## ‚ú® Key Features

The portfolio website includes several modern and interactive features built using JavaScript, HTML, and CSS:

* **Dark/Light Theme Toggle:** Allows users to switch between themes, with the preference saved to local storage.
* **Dynamic Hero Section:** Features a text typing effect that cycles through titles like "Web Developer," "Frontend Enthusiast," and "UI/UX Designer".
* **Animated Particles Background:** Uses the `particles.js` library for a dynamic, engaging background effect.
* **Scroll Progress Bar:** Visually indicates the user's progress through the page.
* **Reveal Animations:** Content is smoothly revealed as the user scrolls down the page.
* **Interactive "Hire Me" Modal:** A responsive form that includes basic validation and simulates a network submission delay with success/error feedback.
* **Custom Cursor:** Enhances the site's unique aesthetic with a custom cursor and glow effect.
